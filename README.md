# Alem Contract - Лендинг компании

Одностраничный лендинг для компании, занимающейся офисной мебелью и комплексным оснащением офисов.

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик и dev-сервер
- **Express.js** - Backend API сервер
- **Prisma + SQLite** - ORM и база данных
- **Чистый CSS** - Стилизация без фреймворков

## Установка и запуск

### 1. Установите зависимости:
```bash
npm install
```

### 2. Инициализируйте базу данных:
```bash
# Генерация Prisma Client
npx prisma generate

# Создание миграции и базы данных
npx prisma migrate dev --name init
```

### 3. Запустите проект:

**Вариант 1: Запуск фронтенда и бэкенда одновременно (рекомендуется)**
```bash
npm run dev:all
```

**Вариант 2: Запуск отдельно**
```bash
# Терминал 1 - Фронтенд (http://localhost:3000)
npm run dev

# Терминал 2 - Бэкенд (http://localhost:3001)
npm run server
```

## Функционал

✅ **Навигация:**
- Плавная прокрутка по якорям
- Фиксированная шапка при скролле
- Мобильное меню (бургер)

✅ **Адаптивность:**
- Мобильные устройства (до 768px)
- Планшеты (768-1024px)
- Десктоп (от 1024px)

✅ **Форма обратной связи:**
- Валидация на фронтенде
- Сохранение заявок в базу данных
- Уведомление об успешной отправке

✅ **Интерактивность:**
- Lightbox для галереи портфолио
- Hover эффекты на карточках
- Плавные анимации

## API Endpoints

- `GET /api/requests` - Получить все заявки (для админки)
- `POST /api/contact` - Создать новую заявку

## База данных

База данных SQLite хранится в `prisma/dev.db`. 

**Модель ContactRequest:**
- `id` - Уникальный идентификатор
- `name` - Имя клиента (обязательно)
- `phone` - Телефон (обязательно)
- `email` - Email (опционально)
- `message` - Сообщение (опционально)
- `createdAt` - Дата создания

## Просмотр заявок

Для просмотра сохраненных заявок можно использовать Prisma Studio:
```bash
npx prisma studio
```

Откроется веб-интерфейс на http://localhost:5555 для просмотра и управления данными.

## Настройка контента

Все тексты, контакты и данные находятся в компонентах. Для изменения:
- Контакты: `src/components/Contacts.jsx`
- Тексты о компании: `src/components/About.jsx`
- Услуги: `src/components/Services.jsx`
- Отзывы: `src/components/Reviews.jsx`
- И т.д.

## Сборка для продакшена

```bash
npm run build
```
